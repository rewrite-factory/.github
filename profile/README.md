## Что делает Рерайт-завод

Редакция региональной газеты или новостного портала получает десятки инфоповодов в день из федеральных источников. Нужно переписать, адаптировать к своей аудитории, сохранить голос издания — и сделать это быстро.

Рерайт-завод автоматизирует этот процесс:

- **Принимает** исходный текст новости (федеральной, пресс-релиза, агентства)
- **Анализирует** стилевой профиль издания (обученный на реальных публикациях)
- **Генерирует** готовый рерайт в стиле редакции
- **Проверяет** факты и флагирует потенциальные ошибки

## Технический стек

- **LLM:** GPT-4o + fine-tuning на стилевых профилях изданий
- **Embeddings:** векторное представление «голоса» редакции
- **Delivery:** веб-интерфейс + API
- **Инфраструктура:** Docker / VPS, CI/CD через GitHub Actions

## Часто задаваемые вопросы

**Q: Чем Рерайт-завод отличается от ChatGPT?**  
A: ChatGPT генерирует универсальный текст. Рерайт-завод обучен на публикациях конкретного издания и воспроизводит его стиль — читатели не замечают разницы.

**Q: Это законно? Авторские права?**  
A: Рерайт — это журналистская практика с многолетней историей. Мы автоматизируем процесс, который редакции и так делают вручную. Текст создаётся новый, оригинальный.

**Q: Как быстро сервис обучается под стиль издания?**  
A: Базовый профиль — за 3–5 дней на выборке публикаций редакции. Точная настройка — 2 недели пилота.

**Q: Подходит ли для Яндекс.Новостей и Google News?**  
A: Да. Тексты уникальные, соответствуют требованиям агрегаторов.

**Q: Что если AI ошибётся?**  
A: Все материалы проходят через редактора как черновик. Система флагирует подозрительные факты. Финальное решение — за журналистом.
